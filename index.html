Smart Markdown Extended Toolbar

Transform your markdown editing experience with a toolbar that adapts to your context and workflow.

Context-Aware

Buttons appear and hide based on your cursor position. Table tools when you're in tables, task toggles on task lines.

Fully Customizable

Drag and drop to build your perfect toolbar. Choose from 20+ markdown tools or use smart presets.

Extension Integration

Works seamlessly with Markdown All in One, markdownlint, and other popular extensions.

Performance Optimized

Smart caching and debouncing ensure smooth performance even in large documents.

Fallback Support

Built-in formatting commands work even without external extensions installed.

Theme Adaptive

Uses VS Code's native Codicons and theming for a consistent, professional look.

Installation

1

Open VS Code

Launch Visual Studio Code on your system

2

Extensions Panel

Press Ctrl+Shift+X to open extensions

3

Search & Install

Search "Markdown Extended Toolbar" and click Install

4

Start Writing!

Open any .md file and see the toolbar in your status bar
